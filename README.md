# Projek POS Flutter

Aplikasi Point of Sale (POS) sederhana yang dibuat menggunakan Flutter.

## 📥 Clone Repository

Untuk mengambil project dari GitHub ke komputer:

```
git clone https://github.com/UmrFaruq/projekfloo.git
cd projekfloo/pos_mobile
```

## ▶️ Menjalankan Project Flutter

Pastikan Flutter sudah terinstall.

1. Install dependencies:

```
flutter pub get
```

2. Jalankan aplikasi:

```
flutter run
```

## 🔄 Update Project (Pull)

Jika ada perubahan dari repository:

```
git pull origin main
```

## 📤 Upload Perubahan (Push)

Jika sudah melakukan perubahan pada project:

```
git add .
git commit -m "update fitur"
git push origin main
```

## 👥 Kolaborasi

Jika ingin berkontribusi pada project ini, silakan clone repository dan lakukan perubahan kemudian push ke branch `main`.
